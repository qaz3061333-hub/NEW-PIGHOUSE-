### 建議 title

價格｜馬爾濟斯

### 建議 category

price_quote_dog

### 建議 content

類型：犬種美容價格

品種：馬爾濟斯

體重區間 1：

- 洗澡基礎價：500
- 大美容基礎價：1100
- 包月基礎價：1800

- 洗澡對客報價：500～750
- 大美容對客報價：1100～1350
- 包月對客報價：1800～2050

體重區間 2：
- 體重：3.1～4.5kg
- 洗澡基礎價：550
- 大美容基礎價：1200
- 包月基礎價：2000
- 體型 / 長毛 / 激動加價：50～250
- 手修加價：200～400
- 洗澡對客報價：550～800
- 大美容對客報價：1200～1450
- 包月對客報價：2000～2250

體重區間 3：
- 體重：4.5kg 以上
- 洗澡基礎價：650
- 大美容基礎價：1300
- 包月基礎價：2200
- 體型 / 長毛 / 激動加價：50～250
- 手修加價：200～400
- 洗澡對客報價：650～900
- 大美容對客報價：1300～1550
- 包月對客報價：2200～2450

包月內容：
- 4 次洗澡含 1 次精緻美容
- 期限 45 天
- 加韓國低弱敏皮毛養護
- 韓國低弱敏皮毛養護未明確寫數量時，預設只送一次

報價提醒：
- 報價前至少需要知道品種與體重。
- 對客應報價格區間，不報單一死價格。
- 實際價格會依體型、毛長、激動或攻擊程度調整。
- 洗澡與大美容已包含基本美容項目，不重複收剪指甲、磨指甲、清耳、拔 / 剃耳毛等基本項目費用。
- 若有嚴重打結、廢毛、體外蟲，需依共通規則或現場評估。
- 若需要腳柱、屁屁、領巾、全身手剪等手修，才另外加入手修加價。
- 手修不是一般洗澡或大美容的預設項目；只有客人明確提出手修需求時，才提醒手修另計或手修加價區間。
- 價格以現場評估與最新 active Knowledge Base 為準。

## 2. 加購｜單項基本美容服務

### 建議 title

加購｜單項基本美容服務

### 建議 category

price_quote_addon

### 建議 content

類型：單項基本美容服務

重要前提：
- 洗澡與大美容已包含基本美容項目。
- 若已做洗澡或大美容，不重複收剪指甲、磨指甲、清耳、拔 / 剃耳毛等基本項目費用。
- 若沒有洗澡或大美容，只單做某項服務，才用單項計費。

剪指甲：
- 價格：100～200
- 一般可先報 100 起。
- 若寵物激動、不配合或有攻擊傾向，最高可能 200。
- 若過度激動或攻擊風險太高，現場可能無法承接。
- 回覆時不要保證一定可以做，需保留現場評估空間。

磨指甲：
- 價格：50～100
- 一般可先報 50 起。
- 若寵物激動、不配合或有攻擊傾向，最高可能 100。
- 若過度激動或攻擊風險太高，現場可能無法承接。
- 回覆時不要保證一定可以做，需保留現場評估空間。

拔耳毛：
- 價格：50～100
- 可單做。
- 洗澡 / 大美容已包含耳朵清潔相關基本處理，不重複加收。
- 若客人描述紅腫、發炎、疼痛、流膿、異味嚴重、疑似感染、一直抓耳朵等嚴重耳況，一律建議就醫。
- 嚴重耳況不由美容服務承諾處理，也不由系統自動判斷醫療問題。

清耳朵：
- 價格：50
- 可單做。
- 洗澡 / 大美容已包含，不重複加收。
- 嚴重耳況一律建議就醫，不由美容服務承諾處理。
- 若客人描述紅腫、發炎、疼痛、流膿、異味嚴重、疑似感染、一直抓耳朵等情況，不要把清耳朵 50 當成解法。

其他基本美容單項：
- 內容範例：清除肛門腺、剃腳底毛、剃肚子毛等。
- 價格：50
- 若不確定是否屬於基本美容單項，需人工確認。

共通提醒：
- 單項價格只適用於沒有洗澡或大美容、只單做某項服務的情況。
- 洗澡與大美容報價中，不要重複加入剪指甲、磨指甲、清耳、拔 / 剃耳毛等基本項目費用。
- 寵物激動、不配合或有攻擊傾向時，實際是否可做與價格需依現場評估。
- 若涉及醫療疑慮、嚴重耳況、皮膚紅腫、發炎、傷口、嚴重敏感，應建議就醫或人工處理。
- 價格以現場評估與最新 active Knowledge Base 為準。

### 測試問題

1. 只剪指甲多少？
2. 只磨指甲多少？
3. 狗狗很兇可以剪指甲嗎？
4. 洗澡還要另外收剪指甲嗎？

- 嚴重耳況建議就醫，不報清耳朵 50 當解法。

## 3. 規則｜報價共通提醒與高風險邊界

### 建議 title

規則｜報價共通提醒與高風險邊界

### 建議 category

price_quote_policy

### 建議 content

類型：報價共通提醒與高風險邊界

報價共通原則：
- 報價前至少需要品種 / 類型與體重。
- 對客應報價格區間，不報單一死價格。
- 價格會依體型、毛長、激動或攻擊程度調整。
- 若缺品種 / 體重 / 膚質 / 分店資訊，應先追問，不要猜。
- 價格以現場評估與最新 active Knowledge Base 為準。
- 價格與規則不應寫死在 code、API、prompt 或 Gemini 記憶中。

洗澡 / 大美容基本項目：
- 洗澡與大美容已包含基本美容項目。
- 包含修剪指甲及磨圓、修剪腳緣、修剃腳底毛、肛門毛、肚子毛、生殖器毛、擠肛門腺、視情況修眼周雜毛、雜毛修整、廢毛梳整、耳朵清潔。
- 若客人有特別不要做的基本項目，需主動提出。
- 單做某項服務才用單項計費。
- 洗澡與大美容報價中，不要重複加入剪指甲、磨指甲、清耳、拔 / 剃耳毛等基本項目費用。

另計規則：
- 另計不是一律建立人工任務。
- 另計代表不可自動算完整總價。
- 可以先回覆可確定的基礎價格，再提醒另計項目需現場評估。
- 手修標示另計時，先報大美容可確定區間，再提醒手修現場另計。
- 手修不是一般洗澡 / 大美容預設項目，只有客人明確提到全身手修、部分手修、腳柱、屁屁、領巾、手剪造型等需求時，才加入手修邏輯。

體外蟲 / 廢毛 / 打結：
- 發現活體成蟲，一律 +150 改為除蚤洗劑。
- 超過 10 隻成蟲，收環境清潔費 300～800。
- 蟲量過多可退貨，並收取已完成項目費用。
- 廢毛與打結不提供固定加價範圍，只提醒現場評估另計。

高風險邊界：
以下情況不可用價格 KB 自動完整回答，應走人工流程或建議就醫：
- 客訴
- 價格爭議，例如「為什麼你們這麼貴」
- 退款
- 受傷
- 流血
- 醫療疑慮
- 嚴重耳況，例如紅腫、發炎、疼痛、流膿、異味嚴重、疑似感染、一直抓耳朵
- 皮膚紅腫、發炎、傷口、嚴重敏感
- 攻擊風險過高、可能無法安全承接

回覆方向提醒：
- 缺資料時先追問，不要猜價。
- 可確定的基礎價格可以先回覆，另計項目需提醒現場評估。
- 體外蟲可以說明 +150 除蚤洗劑、超過 10 隻成蟲環境清潔費 300～800、蟲量過多可能退貨並收已完成費用。
- 嚴重耳況、皮膚嚴重狀況、流血、受傷、退款、客訴、價格爭議，不應用價格 KB 自動完整回答。

- 缺資料時追問，不猜價。

- 體外蟲能說明規則。
- 流血、退款、價格爭議、嚴重耳況不得自動完整回答，應走人工流程或建議就醫。

5. 若這 3 篇測試通過，再建立下一批：貴賓、黃金、貓咪分類。
