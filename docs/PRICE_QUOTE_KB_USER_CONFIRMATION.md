# Price Quote KB User Confirmation v1

資料日期：2026-05-23

本文件適用於 NEW-PIGHOUSE- / new-pighouse-pjdh 專案。此專案目前仍是沙盒 / MVP / 內部流程驗證階段，不是正式 LINE 上線階段。

## 1. 文件目的

本文件是給使用者確認報價規則用。

本文件不是正式價格文件。不是 KB 匯入檔。不是 SQL。不是程式。也不是正式 LINE 自動報價規則。

本文件的目的只是先讓使用者確認以下幾件事：

- 單做項目是否與洗澡 / 大美容內含項目分開
- 另計是否代表轉人工
- 貴賓包月第三級距到底如何算
- 手修完整報價公式是否正確
- 打結 / 廢毛 / 體外蟲是否只提醒另計
- 鬥牛犬類包月贈送規則
- 貓咪巨型貓種是否一律另計

確認完成前，不應把 Price Quote KB Seed Draft v1 當成正式可自動報價資料。

## 2. 使用者回答方式

請使用者依照每個問題直接回答選項即可。

建議回答格式：

```text
問題 1：A
問題 2：A
問題 3：C，這裡請改成......
```

每個問題都會包含：

- 問題編號
- 確認問題
- 目前理解
- 請使用者回答選項
- 如果選項都不對，請使用者補充

如果選項都不符合實際規則，請選「其他，請補充」，並用白話補充正確規則。

## 3. 確認問題 1：剪指甲 / 磨指甲 / 拔耳毛等加價項目

問題編號：問題 1

確認問題：

目前價目表寫洗澡 / 大美容已包含剪指甲、磨指甲、剃腳底毛 / 肚子毛 / 肛門毛、修腳緣、微修頭部面部、擠肛門腺。

但加價項目表中也有剪指甲、磨指甲、拔耳毛等單項價格。

目前理解：

這些單項價格應該是「單獨做」或「住宿外加」時使用，不是在洗澡 / 大美容報價中再加一次。

請使用者回答選項：

A. 是，單項價格只用於單做或住宿外加，不用在洗澡 / 大美容內再加一次。

B. 否，洗澡 / 大美容有些情況仍會再加。

C. 其他，請補充。

如果選項都不對，請補充：

請說明哪些項目是洗澡 / 大美容內含，哪些情況才需要另外加價。

## 4. 確認問題 2：「另計」的處理方式

問題編號：問題 2

確認問題：

價目表中寫「另計」的項目，未來系統應該如何處理？

目前理解：

價目表中寫「另計」的項目，未來系統不應自動報價，應轉人工或現場評估。

請使用者回答選項：

A. 是，另計一律不可自動報價，要轉人工 / 現場評估。

B. 否，有些另計仍可以給初步區間。

C. 其他，請補充。

如果選項都不對，請補充：

請說明哪些「另計」可以先給範圍，哪些一定要人工或現場評估。

## 5. 確認問題 3：貴賓包月第三級距

問題編號：問題 3

確認問題：

貴賓（手吹）包月第三級距到底要依哪個價格規則計算？

目前理解：

目前 seed draft 內有標記疑點：貴賓（手吹）表格中 L 包月為 2500。

但原價目表下方包月範例似乎寫到「6-8.9 即報 2300～2550」，可能與表格不一致。

請使用者回答選項：

A. 依表格 L 包月 2500 計算，所以 6～8.9kg 包月對客報價為 2500～2800。

B. 依下方範例 2300～2550。

C. 這是價目表文字錯誤，需要我人工修正。

D. 其他，請補充。

如果選項都不對，請補充：

請說明 6～8.9kg 貴賓（手吹）包月應該報多少，以及加價上限如何算。

## 6. 確認問題 4：手修完整報價公式

問題編號：問題 4

確認問題：

手修完整報價公式是否正確？

目前理解：

手修不是一般報價預設項目。

只有客人提到腳柱、屁屁、領巾、全身手剪、手剪造型等需求時，才加入手修。

目前理解的手修完整報價公式是：

```text
單次手修報價 = 大美容基礎價 + 毛長 / 激動加價 + 手修加價
```

請使用者回答選項：

A. 是，這個公式正確。

B. 不完全正確，手修有其他算法。

C. 手修一律人工報價，不適合自動算。

D. 其他，請補充。

如果選項都不對，請補充：

請說明手修應該怎麼算，或哪些手修可以算、哪些手修要人工報價。

## 7. 確認問題 5：嚴重打結 / 廢毛 / 體外蟲

問題編號：問題 5

確認問題：

嚴重打結、深層清潔廢毛、體外蟲，未來報價時要不要提供加價範圍？

目前理解：

嚴重打結、深層清潔廢毛、體外蟲，目前先只提醒「現場評估另計」，不自動算入報價區間。

請使用者回答選項：

A. 是，只提醒另計，不提供加價範圍。

B. 可以提供大概加價範圍。

C. 依不同情況分開，例如打結可報範圍、體外蟲需人工。

D. 其他，請補充。

如果選項都不對，請補充：

請說明打結、廢毛、體外蟲各自應該怎麼提醒或報價。

## 8. 確認問題 6：法鬥 / 鬥牛犬類包月贈送規則

問題編號：問題 6

確認問題：

法鬥 / 鬥牛犬類包月多送洗劑或潔膚油的規則，適用範圍是什麼？

目前理解：

目前 seed draft 寫到法鬥包月有特殊贈送項目。

原價目表也寫「鬥牛犬類包月都多送 1 次洗劑 or 潔膚油」。

請使用者回答選項：

A. 這個規則適用所有鬥牛犬類。

B. 只適用法鬥。

C. 依犬種或方案不同，需要人工確認。

D. 其他，請補充。

如果選項都不對，請補充：

請說明哪些犬種或哪些包月方案適用這個贈送規則。

貓咪巨型貓種是否一律另計，不自動報價？
