## 二、重要前提：基本美容項目已包含在洗澡 / 大美容內

洗澡與大美容都已包含以下基本美容項目：

- 修剪指甲及磨圓
- 修剪腳緣
- 修剃腳底毛
- 修剃肛門毛
- 修剃肚子毛
- 修剃生殖器毛
- 擠肛門腺
- 視情況修開眼周雜毛
- 雜毛修整、廢毛梳整
- 耳朵清潔，包含拔 / 剃耳毛、清耳液清潔耳朵內部

如果客人做的是洗澡或大美容，不應再重複收基本美容項目費用。

如果客人有特別不要做的基本項目，需主動提出。

如果客人沒有做洗澡或大美容，只單做某項服務，才使用單次 / 單項計費。

## 三、分類方式

A. 單項基本美容服務

適用：沒有洗澡 / 大美容，只單做某項服務。

B. 美容外加 / 洗劑 / 皮毛護理

適用：客人加購洗劑、草本泥、火山泥、潔膚油、護理套組等。

C. 洗前護理

適用：洗澡前局部清潔、去油、潔膚。

D. 體外蟲 / 環境清潔規則

適用：發現活體成蟲或蟲量過多。

E. 住宿外加項目

適用：住宿期間加散步、租插座、租用或購買睡墊等。
